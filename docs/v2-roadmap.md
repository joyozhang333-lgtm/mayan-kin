# mayan-kin v2 Roadmap

`mayan-kin v2` 的目标，不再只是“会算 Kin 的 skill”，而是升级成一个 **可验证、可解释、可集成** 的玛雅天赋开源基础设施。

## v0.2 基础能力强化

- 扩充公开样本与测试矩阵
- 提升 README、SEO、runtime adapter 与示例
- 新增个人说明书 / 报告输出
- 明确 Dreamspell / Tzolkin 路线边界

## v0.3 解读引擎升级

- 引入更完整的事业 / 关系 / 成长解释模板
- 做“卡点 -> 边界 -> 行动建议”桥接
- 增加更适合咨询、内容产品和 AI 对话的输出结构
- 扩展流年、合盘、波符的更强解释层

## v0.4 体系深度扩展

- 新增神谕十字 / 地球家族等专题
- 增加更多关系动力学与年度趋势输出
- 支持更多报告风格和 runtime 参数化

## v1.0 开源基础设施版本

- 稳定 API / CLI / JSON 契约
- 完整知识库和解释层
- 多运行时适配与文档站
- 让个人用户、咨询师、创作者、AI agent 都能直接复用
