# 流年运势分析方法 (Yearly Fortune Analysis)

## 概述

玛雅天赋的流年分析基于一个核心概念：**你每年生日那天都有一个新的星系印记**。这个年度星系印记代表了你这一年（从生日到下一个生日）的主题能量。

## 计算方法

1. 计算你某一年生日当天的Kin号
2. 从该Kin号推出图腾和调性
3. 排出该年的五大天赋盘
4. 将流年盘与本命盘进行对比

## 流年调性的年度主题

每年的调性代表了你这一年的行动主题：

| 流年调性 | 年度主题 | 关键行动 |
|---------|---------|---------|
| 1 磁性 | 新周期开始年 | 设定新目标，重新定义方向 |
| 2 月亮 | 识别挑战年 | 看清阻碍，面对二元性 |
| 3 电力 | 激活服务年 | 建立连接，服务他人 |
| 4 自存 | 建构形式年 | 搭建结构，落实计划 |
| 5 超频 | 收集力量年 | 聚集资源，扩大影响 |
| 6 韵律 | 恢复平衡年 | 调整节奏，重新平衡 |
| 7 共振 | 内在调频年 | 倾听直觉，灵性成长 |
| 8 银河 | 整合一致年 | 知行合一，内外对齐 |
| 9 太阳 | 实现意图年 | 全力行动，梦想成真 |
| 10 行星 | 完美显化年 | 追求卓越，产出成果 |
| 11 光谱 | 释放解放年 | 放下旧有，简化生活 |
| 12 水晶 | 合作分享年 | 团队协作，分享所得 |
| 13 宇宙 | 超越完成年 | 完成周期，准备下一阶段 |

## 流年图腾与本命盘的互动

### 流年图腾落在本命的哪个位置？

这是流年解读最关键的部分：

**流年图腾 = 本命主印记的图腾**
- 回归之年。这一年你最像"自己"，核心天赋被强化
- 适合做代表自我的重大决定

**流年图腾 = 本命支持位的图腾**
- 轻松支持之年。感觉事情比较顺利，有助力
- 适合巩固已有成果，接受帮助

**流年图腾 = 本命引导位的图腾**
- 方向年。会有明确的指引出现
- 适合做长期规划，跟随灵感

**流年图腾 = 本命挑战位的图腾**
- 成长突破之年。可能有压力和不适，但成长最大
- 适合面对恐惧，走出舒适区

**流年图腾 = 本命隐藏推动位的图腾**
- 潜能觉醒之年。过去未被发现的能力会浮现
- 适合尝试新事物，信任直觉

**流年图腾与本命无直接连接**
- 新能量体验之年。引入全新的频率
- 适合开放心态，学习新的可能性

### 流年调性与本命调性的关系

| 关系 | 含义 |
|------|------|
| 流年调性 = 本命调性 | 共振年，能量加倍 |
| 流年调性 + 本命调性 = 14 | 互补年，内在深层工作 |
| 流年调性 = 本命调性 ± 1 | 过渡年，自然递进 |
| 流年调性 与 本命调性差距大 | 新频率年，需要适应 |

## 流年颜色的能量季节

每年的流年颜色代表不同的能量季节：

- **红色年**: 启动、行动、开创新事物
- **白色年**: 净化、精炼、内在反思
- **蓝色年**: 转化、深度改变、蜕变
- **黄色年**: 收获、成熟、智慧绽放

## 13年大周期

因为调性有13个，所以每13年会经历一个完整的调性周期：

```
第1年: 磁性（开始新周期）
第2年: 月亮（面对挑战）
第3年: 电力（激活连接）
...
第13年: 宇宙（完成周期）
第14年: 新的磁性（下一轮周期）
```

了解自己在13年周期中的位置，可以更好地理解当前阶段的意义。

## 流年与事业

- **红色年 + 磁性/电力/太阳**: 最适合创业、开展新项目
- **白色年 + 韵律/银河**: 适合内部优化、提升质量
- **蓝色年 + 光谱/宇宙**: 适合转型、结束旧项目
- **黄色年 + 行星/水晶**: 适合收获成果、扩大合作

## 流年与情感

- **红色年**: 可能有新的邂逅或关系中的新开始
- **白色年**: 适合净化关系、释放旧模式
- **蓝色年**: 关系中可能有深层的转化和改变
- **黄色年**: 关系成熟、收获承诺或深度的连接
