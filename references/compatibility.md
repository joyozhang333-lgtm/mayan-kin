# 关系动力学（Compatibility）

合盘在 `mayan-kin` 里不是用来判断“合不合适”的裁判工具，而是用来读清楚两个人之间到底在发生什么。
更准确地说，它是一个关系动力学入口：看见支持、张力、互补、镜像和成长方向。

## 先定一个原则

这套体系不输出“你们一定合”或“你们一定不合”。
它输出的是：

- 这段关系的基调是什么
- 谁会支持谁
- 谁会触发谁的课题
- 两个人在一起时最可能生成什么共同能量

这样写，才更适合咨询、内容产品和真实的人际场景。

## 关系动力学看什么

### 1. 位置关系

先看 B 的图腾落在 A 的哪一层，再反过来看 A 落在 B 的哪一层。

常见关系类型可以读成：

- 主印记相同：镜像和共鸣
- 支持位落点：天然支持和舒适
- 引导位落点：方向感和启发
- 挑战位落点：张力、摩擦、成长
- 隐藏推动位落点：深层吸引和潜意识触发
- 五大位之外：新频率学习，双方都要更多耐心

关键不是“落在哪”本身，而是这个落点会把关系推向哪种互动模式。

### 2. 颜色互动

颜色层更像关系的底色。

| 颜色组合 | 关系底色 | 常见体验 |
|---------|---------|---------|
| 红 + 白 | 启动与净化 | 一个推、一个理，互补明显 |
| 蓝 + 黄 | 转化与成熟 | 一个变、一个收，容易一起升级 |
| 红 + 蓝 | 启动与转化 | 张力大，推动快，变化也大 |
| 白 + 黄 | 净化与成熟 | 容易一起看清问题、收敛方向 |
| 同色相遇 | 强共振 | 很容易懂彼此，也容易放大彼此 |

颜色不是给关系贴标签，而是帮你判断这段互动更偏“稳定、转化、推进还是收束”。

### 3. 调性互动

调性读的是节奏，不是性格好坏。

| 关系 | 互动特征 |
|------|---------|
| 同调性 | 节奏相近，容易同步 |
| 调性相邻 | 一个带一个，比较自然 |
| 调性互补 | 彼此补位，但需要适应 |
| 调性差距大 | 需要主动调频和沟通 |

节奏不合，并不等于关系差，只代表你们需要更明确的协作方式。

### 4. 合盘 Kin

两个人的 Kin 组合后，可以得到一个“关系 Kin”。
这个 Kin 代表的是：

- 你们在一起时会创造什么样的共同场域
- 这段关系更像推动、修复、学习，还是收成
- 你们共同的主题是什么

合盘 Kin 适合用来写关系摘要，而不是拿来做绝对判断。

### 5. 交叉位置

交叉位置是最适合写“关系建议”的地方。

例如：

- A 的支持位正好是 B 的挑战位，说明 A 可能天然有帮助 B 的能力，但也会触发 B 的成长压力
- A 的引导位正好是 B 的隐藏推动位，说明两人之间常有深层共振和方向牵引
- 两人共享相近的引导主题，说明这段关系很适合一起朝一个方向走

这一层最适合落到“你们为什么会这样互动”。

## 常见关系类型

### 伴侣 / 情侣

重点不是舒不舒服，而是能不能长期共同成长。

建议优先看：

- 位置关系
- 调性互动
- 合盘 Kin 的主题

如果两个人互为支持位，通常更稳定；如果互为挑战位，通常更有成长力，但也更需要沟通和边界。

### 事业伙伴

事业伙伴更适合看：

- 图腾互补性
- 节奏是否协调
- 是否能互相激发而不是互相拖慢

好的事业伙伴，不一定最温柔，但一定能让事情更清楚、更推进。

### 亲子关系

亲子关系里，重点是理解角色而不是贴好坏标签。

- 父母在孩子支持位：孩子容易从中获得稳定
- 父母在孩子引导位：孩子容易把父母当成方向源
- 父母在孩子挑战位：会有压力，但也常带来深刻成长

### 朋友关系

朋友关系更适合看：

- 颜色共振
- 调性是否好沟通
- 彼此是不是同类成长路径

朋友不一定要完全一样，但应该能让彼此更清楚。

## 适合回答的问题

- 为什么我们一见面就有熟悉感
- 为什么这段关系很有吸引力但也很累
- 为什么我总在这段关系里被激发
- 为什么有些合作很顺，有些总卡
- 这段关系更适合长期、阶段性，还是学习型

## 适合输出的场景

- 情侣合盘
- 事业伙伴分析
- 亲子关系分析
- 朋友关系说明
- 咨询中的关系追问
