# 玛雅天赋解读技能 (Mayan Kin Skill)

基于卓尔金历 (Tzolkin) / Dreamspell 体系的专业星系印记解读系统。通过阳历出生日期推算个人天赋图腾与调性，解读五大天赋力量，提供流年运势分析与双人合盘。

## 使用时机

当用户请求以下类型的帮助时，应启用此技能：

1. **星系印记查询**：请求计算玛雅天赋、星系印记、Kin号
2. **天赋解读**：询问图腾、调性、五大天赋盘含义
3. **流年分析**：询问某个年份/时期的能量与运势
4. **合盘分析**：两人之间的天赋互动与关系解读
5. **天赋运用**：如何在生活、事业、情感中运用天赋
6. **关键词触发**：「玛雅天赋」「星系印记」「Kin」「图腾」「调性」「卓尔金历」「Dreamspell」「玛雅合盘」

## 核心功能

- Kin号计算（阳历生日 → 星系印记）
- 五大天赋盘完整排盘与解读
- 20图腾 × 13调性深度解析
- 流年运势分析（每年的星系印记与能量）
- 双人合盘（关系互动解读）
- 天赋运用指导（事业、情感、成长路径）
- 波符（Wavespell）与颜色能量解读
- 深度版解读校准：个人、流年、合盘报告的触发条件、误读风险、验证问题、最小实验
- 现实表达校准：把五大位置翻译成公开表达标签、适配场域和现实验证问题
- 科学验证实验工具：生成盲测包、分离答案 key、按正确率/随机基线/p 值计算科学准确率分数
- 公开人物 holdout：1425 条 Wikidata 样本、冻结评分协议、train/dev/holdout 划分和失败结果记录
- 七维公开传记标签：1080 条 Wikipedia/Wikidata 公开人物细标签、v2 holdout 与时间切分评估
- 前瞻预测登记：先锁定预测，再用未来证据回填评分

## Kin号计算

### 核心算法

使用 `scripts/mayan_calc.py` 计算，核心逻辑：

1. **参考日期**：2013年7月26日 = Kin 164（黄银河种子）
2. **计算天数差**：从参考日期到目标日期的天数
3. **跳过闰日**：Dreamspell体系中，2月29日不计入Kin编号（"超越时间日"）
4. **取模**：Kin = (164 + 天数差 - 跳过的闰日数) % 260，若为0则为260

### 从Kin号推导图腾和调性

```
调性 = ((Kin - 1) % 13) + 1
图腾 = ((Kin - 1) % 20) + 1
```

## 五大天赋盘

### 盘面结构

```
          【引导】
           ↑
【支持】← 【主印记】→ 【挑战】
           ↓
        【隐藏推动】
```

### 各位置含义

**1. 主印记（中心）— 你的核心天赋**
- 由出生日期直接算出
- 代表你这一生最核心的能量与使命
- 是你天生自带的频率，别人一眼就能感受到的你
- 好比你的"出厂设置"，是你最自然的表达方式

**2. 支持/类似（右方）— 你的天然武器**
- 公式：主图腾编号 + 支持图腾编号 = 19（超过20减20）
- 调性：与主印记相同
- 这是你最容易发挥的辅助力量，像你的右手一样自然
- 它支持你完成主印记的使命，给你力量加持
- 在你感到迷茫时，回到支持位的能量会让你安心
- **为什么是"支持"**：它与你的主印记颜色互补（红↔白、蓝↔黄），代表阴阳互补的和谐力量

**3. 引导（上方）— 你的人生导航**
- 与主印记同色（同一个颜色家族的图腾）
- 调性：与主印记相同
- 具体哪个同色图腾，由调性决定（见引导图腾计算规则）
- 这是你的"高我"力量，指引你前进的方向
- 当你做重大决策时，引导位的能量会来帮助你
- **为什么是"引导"**：同色=同一族群=更高版本的自己，是你灵魂进化的方向

**4. 挑战/拓展（左方或下方）— 你的成长功课**
- 公式：主图腾编号 + 10（超过20减20）
- 调性：与主印记相同
- 这不是你的"弱点"，而是你最大的成长空间
- 你对它既陌生又好奇，学会它就能让天赋翻倍
- 它代表你一开始不习惯，但一旦掌握就会带来巨大突破的能力
- **为什么是"挑战"**：它与你的主印记在色环上正对面（相隔10个图腾），代表你需要整合的对立面

**5. 隐藏推动（下方）— 你的潜意识超能力**
- 公式：主图腾编号 + 隐藏推动图腾编号 = 21（超过20减20）
- 调性：14 - 主印记调性
- 这是你潜意识深处的能量，平时看不见但力量巨大
- 当你放下头脑、跟随直觉时，它就会浮现
- 它是你在关键时刻爆发出来的"隐藏技能"
- **为什么是"隐藏推动"**：它的调性与你的调性加起来等于14（超过13的完整周期），代表你内在镜像的力量

### 引导图腾计算规则

| 主印记调性 | 引导位移 | 说明 |
|-----------|---------|------|
| 1, 6, 11 | +0 | 引导就是自己（自我引导） |
| 2, 7, 12 | +12（mod 20） | 往前12个图腾 |
| 3, 8, 13 | +4（mod 20） | 往前4个图腾 |
| 4, 9 | +16（mod 20） | 往前16个图腾（= 往回4个） |
| 5, 10 | +8（mod 20） | 往前8个图腾 |

注意：引导图腾必须与主图腾同色，以上位移确保了这一点。

### 五大天赋盘的成长路径

```
隐藏推动 → 支持 → 主印记 → 挑战 → 引导
  (潜意识)  (舒适区)  (核心)  (成长区)  (目标)
```

**成长路径解读**：
1. **从隐藏推动出发**：先觉察自己潜意识中的力量，认识到自己有些能力是天生的、只是还没被意识到
2. **进入支持位**：用你最熟悉的辅助能力，建立自信和安全感
3. **扎根主印记**：充分发挥核心天赋，这是你在世界上最独特的表达
4. **拥抱挑战位**：有勇气走出舒适区，学习对面的能量，天赋才能翻倍
5. **到达引导位**：当五大力量整合，你就活出了最高版本的自己

## 流年运势分析

### 计算方法

每年的生日当天有一个新的流年Kin，计算方式：
1. 计算该年生日当天的Kin号
2. 排出流年的五大天赋盘
3. 将流年盘与本命盘对比解读

### 解读要点

读取 `references/yearly-fortune.md` 获取完整流年解读方法。

如果是 agent / runtime 环境，先通过 `references/knowledge-index.json` 确认应该加载这张卡，再按需读取正文。

核心关注：
- 流年图腾与本命图腾的关系
- 流年调性代表的年度主题
- 流年与本命五大位置的互动

## 双人合盘

### 合盘方法

读取 `references/compatibility.md` 获取完整合盘解读方法。

如果问题同时涉及团队、关系节奏或深层镜像，可继续按索引补 `oracle.md` 或 `earth-families.md`。

核心分析：
1. **图腾关系**：两人主图腾之间是支持、挑战、引导还是隐藏推动关系？
2. **调性互动**：两人的调性如何互动？是否互补？
3. **颜色互动**：两人的颜色家族关系
4. **交叉分析**：A的主印记出现在B的哪个位置？反之亦然？
5. **合盘Kin**：两人Kin号之和的含义

## 解盘流程

### 第一步：计算星系印记

1. 获取用户的阳历出生日期
2. 运行 `scripts/mayan_calc.py` 计算Kin号
3. 推导图腾和调性

### 第二步：排出五大天赋盘

1. 由主印记推算支持、挑战、隐藏推动、引导
2. 画出天赋盘图示
3. 标注各位置的图腾、调性、颜色

### 第三步：核心解读

1. 主印记的核心天赋与使命
2. 调性代表的生命品质
3. 五大位置各自的深度含义
4. 整体天赋蓝图的故事线

### 第四步：实用指导

1. 事业方向：哪些领域最适合发挥天赋？
2. 情感关系：天赋在关系中如何表达？
3. 成长建议：当前阶段应该关注哪个天赋位？
4. 日常实践：如何在日常生活中连接天赋能量？

### 第五步：流年分析（如需要）

1. 计算当前流年Kin
2. 解读流年与本命的互动
3. 给出年度行动建议

## 常见问题处理

- **只知道农历生日** → 提示需要转换为阳历，建议使用万年历工具
- **2月29日出生** → 在Dreamspell体系中，2月29日是"超越时间日"(Day Out of Time的特殊版本)，该日出生者通常使用2月28日或3月1日的Kin
- **对结果不认同** → 天赋是潜能，不一定已经显化；有些天赋需要主动开发
- **问"准不准"** → 玛雅天赋不是预测，是一面自我认知的镜子，关键在于共鸣和启发
- **想知道"好不好"** → 每个印记都有独特价值，没有好坏之分，只有是否被充分运用
- **请求计算他人印记** → 可以计算，但建议解读时以帮助理解关系为主

> 「In Lak'ech（我是另一个你）」
>
> 玛雅天赋的核心理念：每个人都是独特的宇宙印记，认识自己的天赋，就是找到与宇宙共振的频率。
