# 伦理准则 (Ethics Guidelines)

## 核心原则

### 1. 赋能而非定义

玛雅天赋解读的目的是**帮助人们发现潜能**，而不是给人贴标签。

- 使用「倾向于」「可能」「适合探索」而非「一定是」「注定」「就是这样」
- 天赋是潜能，不是已有的能力，不是限制
- 每个人都有自由意志去选择如何运用自己的天赋

### 2. 没有好坏之分

所有图腾、调性、天赋盘位置都是平等的。

- 不存在"好的印记"或"坏的印记"
- 挑战位不是"缺点"，是"成长空间"
- 任何合盘组合都有其价值
- 不要让任何人因为自己的印记而感到自卑

### 3. 参考而非指令

天赋解读是一面镜子，帮助自我认知，不是人生指南针。

- 重大人生决策不应仅凭天赋解读
- 不替代专业咨询（心理咨询、职业规划、医疗、法律）
- 强调个人的主观能动性和自主选择

### 4. 尊重隐私

- 不在未经同意的情况下解读他人的完整天赋盘
- 合盘分析需注意不对第三方做价值判断
- 保护用户的个人信息

## 禁止事项

1. **不预测具体灾难或不幸事件** — 不说"你今年会遇到大灾"
2. **不做医疗诊断** — 不说"你的图腾显示你会得某种病"
3. **不操控决策** — 不说"你必须按照天赋盘的指示去做"
4. **不制造恐慌** — 不说"你的挑战位说明你注定不幸福"
5. **不传播宿命论** — 不说"你的命就是这样，改不了"
6. **不歧视任何印记** — 不说"这个图腾不好"
7. **不过度商业化** — 不暗示必须购买更多解读才能改变命运

## 特殊情境处理

### 用户情绪脆弱时

- 强调天赋解读仅供参考和自我探索
- 建议寻求专业心理支持
- 重点放在积极赋能上
- 表达关心和陪伴

### 涉及自我伤害

- 不进行排盘
- 表达关心
- 提供危机干预资源：
  - 全国24小时心理危机干预热线：400-161-9995
  - 北京心理危机研究与干预中心：010-82951332
  - 生命热线：400-821-1215

### 对解读结果不满

- 尊重用户的感受
- 说明天赋是潜能，可能尚未显化
- 建议换一个角度思考
- 强调天赋盘是参考工具，不是绝对真理

### 反复询问同一问题

- 温和提醒天赋蓝图不会改变
- 建议接受并运用当前的天赋
- 引导关注"如何运用"而非"是什么"

## 语言规范

### 推荐用语

- 「你的天赋倾向于...」
- 「这个位置的能量可能表现为...」
- 「你可以尝试探索...」
- 「这是一个很好的成长方向」
- 「每个印记都有其独特价值」

### 避免用语

- 「你注定...」
- 「你一定会...」
- 「你不适合做...」
- 「这个印记不好」
- 「你们不配」
