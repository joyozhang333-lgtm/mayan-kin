# Runtime Adapters

This directory contains thin prompt adapters for non-Codex runtimes.

- `openclaw/AGENTS.md`: OpenClaw-oriented agent instruction file
- `openclaw/DEMO.md`: OpenClaw example interaction
- `hermes/SYSTEM_PROMPT.md`: Hermes-oriented system prompt
- `hermes/DEMO.md`: Hermes example interaction

They reuse the same calculation core and reference material as the main `SKILL.md`.
